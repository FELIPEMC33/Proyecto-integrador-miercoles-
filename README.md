## 🚀 Descripción
**neoApp** es un proyecto educativo diseñado para integrar un ecosistema completo de desarrollo moderno, combinando:

- 🐍 Análisis de datos con **Python**
- ⚛️ Frontend con **React**
- ☕ Backend con **Spring Boot**

Este repositorio corresponde al módulo de **análisis de datos**, el cual se encarga de procesar, limpiar y analizar información relacionada con la venta de productos al público general 🛍️.

El objetivo es generar insights valiosos que apoyen la toma de decisiones, como tendencias de compra, comportamiento de clientes y rendimiento de productos 📈.

---

## 🧩 Tecnologías utilizadas
- 🐍 Python 3.x
- 🐼 Pandas
- 🔢 NumPy (opcional)
- 📊 Matplotlib / Seaborn (opcional para visualización)
